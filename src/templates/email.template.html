How to Control

To control the emulator, reply to this email or send a new email with a maximum of three inputs, in order.

The following keys are available as input:

  • UP, DOWN, LEFT, RIGHT - Direction controls
  • A, B - Action buttons
  • START, SELECT - Menu controls
  • L, R - Shoulder buttons

Example:

UP RIGHT A

Frequently Asked Questions

What should I put in my email?

Just the inputs, nothing else! We'll try to input everything until we find something we don't know.

Do I need to put it in upper/lowercase?

Whatever you prefer. The emulator is case insensitive, so it doesn't matter.

Can I add other text to my email?

No, please don't add any additional text. The system only recognizes the input commands (UP, DOWN, A, B, etc.). Any other text might cause the command to fail.

How do I format multiple commands?

Simply put a space between each command. For example: "UP RIGHT A" will press Up, then Right, then A in sequence.

How many commands can I give?

You can give a maximum of three commands per mail. Any more will be ignored.

What if I just want to see the current state?

You can just send an empty email. You will always get the current state.
